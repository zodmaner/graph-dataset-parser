# Graph-Dataset-Parser

A simple graph dataset parser that parses graph datasets from [[0](https://an.kaist.ac.kr/traces/WWW2010.html)] and [[1](http://snap.stanford.edu/data/index.html)] into a format that is suitable for consumption by [Pregel+](http://www.cse.cuhk.edu.hk/pregelplus/) applications.
